# “原行星盘与行星形成暑期学校及讨论班”会议通知

原行星盘是围绕新生恒星周围的富含尘埃的气体盘，也是行星形成的场所。随着数以千计系外行星的发现，以及高分辨率观测捕捉到原行星盘中普遍存在的各类精细“亚结构”，人们对盘物理和行星形成过程的认识正在发生飞跃。为了推动我国在这一前沿领域的长足发展和人才队伍建设，北京大学科维理天文与天体物理研究所将于2026年7月20日至8月28日举办“原行星盘与行星形成”暑期学校及讨论班。

### 一、 会议基本信息
* **主办单位**：北京大学物理学院，北京大学科维理天文与天体物理研究所
* **活动地点**：北京市海淀区颐和园路5号 北京大学科维理研究所
* **活动时间**：2026年7月20日至8月28日（共六周）
    * 7月20日上午：报到
    * 7月20日 - 24日：暑期学校
    * 7月25日 - 8月28日：讨论班
    * 8月29日：结束
* **面向对象**：天文、物理、行星科学相关专业的高年级本科生和研究生
* **报名截止时间**：2026年5月11日
* **官方网站**：[https://indico.ihep.ac.cn/event/16681/](https://indico.ihep.ac.cn/event/16681/)
* **授课语言**：中英文结合

### 二、 暑期学校（第一周）讲授内容
1.  原行星盘的观测特性和动力学与演化
2.  行星和行星系统的形成
3.  （磁）流体，N体和辐射转移的数值模拟方法及其在本领域中的应用

### 三、 讨论班（二至六周）形式与内容
主讲教师和其他参会教师将提供相关科研课题，与学生进行双向选择。原则上每个学生（或2人小组）选择一个课题。期间安排日常学术交流，并在第六周末进行汇报总结，旨在开拓科研方向并产出成果。
